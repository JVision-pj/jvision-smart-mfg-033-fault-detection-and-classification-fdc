# 故障偵測與分類指揮台

即時監控設備感測數據，自動偵測製程異常並分類故障類型之系統，廣泛用於半導體設備。

- 原始系統：故障偵測與分類系統（Fault Detection and Classification, FDC）
- 產業分類：智慧製造｜製造管理
- 使用部門：設備工程、製程工程
- 工作流程：感測數據擷取 → 模型比對分析 → 異常偵測分類 → 告警通知 → 對策執行驗證
- AI 應用：機器學習建立故障分類模型、AI持續學習優化偵測敏感度
- 主要 KPI：異常偵測準確率、誤報率、平均偵測時間
- 代表產品：PDF Solutions Exensio、Onto Innovation（原Rudolph Technologies）相關FDC方案
